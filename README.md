# Gerador de Currículos em PHP

Sistema completo para criação de currículos profissionais.

## Como usar:
1. Coloque a pasta no XAMPP/htdocs/
2. Acesse http://localhost/gerador-curriculo/
3. Comece criando seu currículo!

## Funcionalidades:
- Formulário multi-etapas
- Campos dinâmicos
- Preview em tempo real
- Download em PDF
